Data Source

All data sourced from PropertyGuru Singapore (propertyguru.com.sg), Singapore's largest property listing portal. Data represents active listings across all 28 URA planning districts.

Collection Method

Automated scraping using a browser-based fetcher (nodriver + Chrome) that navigates PropertyGuru search pages and extracts structured listing data from the site's Next.js data layer. The scraper uses adaptive query splitting (by bedroom count, property type, and price range bisection) to ensure complete coverage within pagination limits.

Coverage

    Key Metrics

    • Median Price: Middle value of all listing prices — more robust than mean for skewed distributions.
    • PSF (Price per Square Foot): Listing price ÷ floor area. The raw PSF from PropertyGuru is unreliable for some listings; we recompute it from price and floor area.
    • Gross Rental Yield: (Median Monthly Rent × 12) ÷ Median Sale Price × 100. This is a rough indicator — actual yields vary by unit and exclude costs.

    Limitations

    • Listing prices are asking prices, not transaction prices — actual sale/rent amounts may differ.
    • PropertyGuru listings may not capture the full market (private sales, other portals).
    • Cloudflare anti-bot protection occasionally blocks scraping, causing temporary data gaps.
    • Detail enrichment (tenure, furnishing, etc.) is a separate phase still in progress; fields from that phase are not shown on this dashboard.
    • Rental yield calculations use district-level medians, not matched unit pairs.
    • Sale and rent are scraped on independent schedules; KPIs reflect each type's most recent full sweep.
